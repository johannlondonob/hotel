URI Método Parámetro Descripción
/api/v1/booking GET Consulta todas las reservas activas.
/api/v1/booking/(id) GET id Consulta una reserva en específico. Se requiere el id de la reserva.
/api/v1/booking POST Se crea una nueva reserva.
/api/v1/booking/(id) PUT id Se actualizan los datos de una reserva en específico. Se requiere el id de la reserva.
/api/v1/booking/(id) DELETE id Se elimina una reserva en específico. Se requiere el id de la reserva.
/api/v1/room GET Consulta todas las habitaciones activas.
/api/v1/room/(id) GET id Consulta una habitación en específico. Se requiere el id de la habitación.
/api/v1/room POST Se crea una nueva habitación.
/api/v1/room/(id) PUT id Se actualizan los datos de una habitación en específico. Se requiere el id de la habitación.
/api/v1/room/(id) DELETE id Se elimina una reserva en específico. Se requiere el id de la reserva.
